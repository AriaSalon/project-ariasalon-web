# Jyllinge Salon – Hjemmeside

React + TypeScript + Tailwind CSS hjemmeside for Jyllinge Salon.

## Kom i gang

```bash
npm install
npm run dev
```

## Tilføj billeder til galleriet

1. Opret mappen `public/gallery/`
2. Kopier dine billeder ind i den mappe og navngiv dem `foto1.jpg`, `foto2.jpg` osv.
3. Opdater arrayet i `src/components/Gallery.tsx` – se kommentarerne i filen.
